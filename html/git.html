Git

  • init-создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий Git.
  • clone -клонирование репозитория из удаленного хранилища на локальную машину.
  • push-выгрузка изменений в удаленный репозиторий.
  • pull-получение изменений из удаленного репозитория на локальную машину.
  • status-просмотр списка измененных файлов.
  • branch -создание ветки.
  • checkout -переход между ветками(переход в ветку).
  • commit-сохранение изенений в Git.
  • log-просмотр истории коммитов в репозитории.
